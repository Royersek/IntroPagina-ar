Experiencia de Realidad Aumentada: Pez 3D

Prototipo educativo en RA basado en biología marina. Parte de los contenidos de ciclo básico, llevado al mundo 3D usando MindAR + A-Frame.

1. Investigación inicial biología

Elegimos trabajar con una especie de pez para explicar características físicas y diversidad marina. La idea nace del videojuego en proceso “Galaga Ocean”.

2. Referencias técnicas y conceptuales fuentes

Inspiración en Galaga (Namco, 1981)

Técnicamente: documentación de MindAR y A-Frame, modelos .glb desde Sketchfab.

3. Diseño de la experiencia RA

Objetivo: que el usuario vea el pez en 3D sobre una imagen y relacione lo que aprendió en biología con su entorno.

Tipo de RA: image tracking con MindAR. Se reconoce el marcador (derivado del Hiro) y se ancla el pez.

4. Desarrollo técnico stack

MindAR para detección + A-Frame para la escena 3D. HTML5, VS Code y GitHub para el proyecto.

Sumergete en la experiencia:

Escaneá este código QR con la cámara de la experiencia de realidad aumentada. Luego escaneá el marcador "Hiro" para que aparezca el modelo 3D del pez.
